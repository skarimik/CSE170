About
  • Event Queuer is an open source project designed to simplify the daily routines of all kinds of individuals. It is based around the concept of simplyfying the way that people schedule events. The use of fliers and bulletin boards to showcase events is very common in many comunities. When passing by these event fliers, many people simply glance over the event and proceed with their day because of responsibilities like being on time to work or class. Even if they see an event that they want to attend, it's quite tedius to input all of the details onto their phone, then schedule it into their personal calendar. Some individuals don't even bother, instead prefering to make a mental note; unfortunately, many, ourselves included, simply forget.

  • Event Queuer is a solution to this user need. With this free and simple web based application, event organizers can generate a QR code that contains information, like time, date, and description, about their event. Then they are able to put this QR code on their fliers. The other user base, the general population, can use this app to scan the QR code of events and all of the information registered into that QR code will be automatically synced into their Google Calendar. This way people can save time and be more organized!
How To Scan

  1. Go to the home page then click on the button with the picture of the phone and QR code.

  2. Scanning Should start by itself, You have 20 seconds to scan your QR code, other wise the scanning process will stop, and you need to restart the scanning process.

  3. When the Scanning is successful, the scanning process will stop automatically, and the result will be shown. If there is any error or you scanned a wrong QR code, you can click on Restart Scanning button and scan the QR code again.

  4. When you are happy with the result of the scann, click on Save button, and that will save the result for you on the Submission Page (the second button on the home page with a picture of task manager), so it will be ready for you to submit it to your Google Calendar.

  5. You will be alerted that your scann result was saved and then you are done.

How To Create a QR Code?

  1. Go to the home page then click on the Plus button.

  2. Fill in the required fields according to the event that you are hosting.

  3. When you are done then click on Create QR button. This will generate a QR code for you that will contain your event's information.

  4. Then print the QR code so you can use it whenever.

  5. You can create a new QR code by just changing the fields and clicking on the Create QR button again.

How To Submit an Event?

  1. Go to the home page then click on the second button with the picture of the Task Sheet.

  2. If you have scanned a QR code preveiously and saved it, the information should be present in this page. If not you can create your own event or edit the saved event here.

  3. When you are ready to submit, just click on Submit button and this will tru to put the event on your Google Calendar.

  4. You will receive an alert saying "Event Create: ..." where the three dots represent the content of the Event that is being created. If the instead of a content you see undefined, that means that there is an issue with the permission that you have granted the website. You need to make sure that the Event_QR web app has access rights to your calendar in order for the web app to be able to submit any events to your Google calendar.

  5. After you see the alert, the event should be present on your calendar.

How To

Once signed in, the application presents the user with three main functions laid out on the screen. The user can scroll up and down the page to access these functions. The first icon, the cell phone, symbolizes the scanning feature. When the user clicks on it, they are prompted to use their camera to scan an available QR code. Once scanned, the application will interpret the QR code and integrate the information into the users calendar.

The second feature, represented by the QR icon, allows the user to generate a QR code for their specified event. The user will be prompted to input information like event title, time, date, location, and discription. After they press generate QR, the interface will present a QR code and the user will be able to screenshot or save the image to their phone. After that, the user can print these codes onto their fliers so others can easily keep track of the event.

The third feature, represented by the calendar icon, is a quick shortcut to the users Google Calendar information. The user can see all of their past and upcoming events in a simple and clean interface.

Common Errors

Google Calendar not working:

Press the 3 dash menu icon on the top right of the home page. They you will be prompted with multiple options through a sidebar menu. Click log out, then refresh the page and log back in.

Not syncing to calendar:

There are a variety of QR codes that do many different things. Some load webpages and others open up different applications. If scanning an arbitrary QR code, and prompted with a error saying that the QR is incompatible, it means that the QR code wasn't generated by Event Queuer, and is theerefore incompatible.

Unable to scan:

Make sure you are on a mobile device and either using Google Chrome, Firefox, or Safari as the browser. Many third party browsers like Opera doesn't contain the proper API to run this web based application.

Popup "Scan Undefined":

Case 1: This error may happen when the user tries to scan a QR code that wasn't created by Event Queuer. If this error appears, check if the QR code is generated from Event Queuer and refresh the page.

Case 2: This error may happen when the user has a rigorous adblocker. Once signed in, the user must give Event Queuer permission to access their Google Calendar. If this popup didn't appear, this means that an adblocker has blocked it from appearing when the user first signs in. To fix this, disable your adblocker, then refresh the page.

Camera not working:

Some browsers rigorously block the websites from using the camera without notifying the user. Go to the top right of the website url and check to see if there is a camera icon with a red line over it. If there is, click it and enable camera permissions.
